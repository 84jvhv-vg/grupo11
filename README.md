# 📊 TeamDashboard - GRUPO 2.1

## 🎯 Descripción
Dashboard de equipo completo con gestión de tareas, miembros, calendario y análisis en tiempo real. Desplegado en **Cloudflare Pages** con almacenamiento local.

## ✨ Características
- 📊 **Dashboard**: Estadísticas en tiempo real, gráficos de progreso, actividad reciente
- ✅ **Tareas**: CRUD completo, prioridades, asignación, filtros
- 👥 **Miembros**: Gestión de equipo, roles, avatares
- 📅 **Calendario**: Visualización mensual con eventos
- 🎨 **Temas**: Claro/Oscuro/Sistema
- 💾 **Persistencia**: Guardado automático en localStorage
- 📱 **Responsive**: Funciona en todos los dispositivos
- 🔄 **Export/Import**: Backup de datos
- 🚀 **CDN Global**: Desplegado en Cloudflare Pages
